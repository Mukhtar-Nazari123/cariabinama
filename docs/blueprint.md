# **App Name**: کاریابی نما

## Core Features:

- جستجوی شغل: به کاربران اجازه می دهد تا بر اساس کلمات کلیدی و مکان به جستجوی شغل بپردازند.
- لیست مشاغل: لیستی از آگهی های شغلی را با جزئیات مربوطه و اطلاعات شرکت نمایش می دهد.
- توصیه های شغلی: با استفاده از یک ابزار، مشاغلی را بر اساس مهارت ها و ترجیحات کاربران پیشنهاد می دهد.
- ثبت نام کاربر: کاربران را قادر می سازد تا یک حساب کاربری ایجاد کرده و تنظیمات برگزیده شغلی خود را ذخیره کنند.
- پروفایل شرکت: به شرکت ها اجازه می دهد تا پروفایل ایجاد کرده و آگهی های شغلی ارسال کنند.
- هشدار های شغلی: اعلان های ایمیلی را در مورد آگهی های شغلی جدیدی که با معیارهای آنها مطابقت دارند، برای کاربران ارسال کنید.
- پست های وبلاگ: مقالات مربوط به ارائه مشاوره شغلی را منتشر کنید.

## Style Guidelines:

- رنگ اصلی: آبی تیره (#1A237E) برای انتقال اعتماد و حرفه ای بودن.
- رنگ پس زمینه: خاکستری روشن (#F5F5F5) برای ظاهری تمیز و مدرن.
- رنگ برجسته: زرد (#FFEB3B) برای برجسته کردن اقدامات و اطلاعات کلیدی.
- متن بدنه: 'PT Sans'، یک sans-serif انسانی برای خوانایی.
- عناوین: 'Space Grotesk'، یک sans-serif که حس و حال مدرن و فن آوری را القا می کند.
- از آیکون های واضح و حرفه ای مربوط به دسته بندی ها و خدمات شغلی استفاده کنید.
- طرح بندی تمیز و شهودی را با تمرکز بر ناوبری آسان و سلسله مراتب واضح حفظ کنید.
- برای بهبود تجربه کاربر، از انتقال ها و انیمیشن های ظریف استفاده کنید.
